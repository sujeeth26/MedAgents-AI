════════════════════════════════════════════════════════│═
          AGENT DETAILS                                 │
════════════════════════════════════════════════════════│═
                                                         │
AGENT 1: QUERY ANALYSIS                                 │
┌─────────────────────────────────────────────────────┐ │
│  Input: User query string                            │ │
│                                                      │ │
│  Processing:                                         │ │
│  1. Analyze query semantics                         │ │
│  2. Classify query type                             │ │
│  3. Extract medical terminology                     │ │
│  4. Generate search keywords                        │ │
│  5. Determine retrieval count (3-10)                │ │
│  6. Assess complexity (simple/moderate/complex)     │ │
│                                                      │ │
│  Output:                                             │ │
│  - query_type: str                                   │ │
│  - key_terms: List[str]                             │ │
│  - search_keywords: List[str]                       │ │
│  - retrieval_count: int                             │ │
│  - complexity: str                                   │ │
│  - reasoning: str                                    │ │
└─────────────────────────────────────────────────────┘ │
                                                         │
AGENT 2: RETRIEVAL                                      │
┌─────────────────────────────────────────────────────┐ │
│  Input: Query analysis + original query              │ │
│                                                      │ │
│  Processing:                                         │ │
│  1. Generate query embedding                        │ │
│  2. Perform Pinecone similarity search              │ │
│  3. Retrieve top-K documents (adaptive)             │ │
│  4. Rank by relevance score                         │ │
│                                                      │ │
│  Output:                                             │ │
│  - List of documents with:                          │ │
│    - content: str                                    │ │
│    - score: float (0-1)                             │ │
│    - metadata: dict                                  │ │
│    - source: str                                     │ │
│    - rank: int                                       │ │
└─────────────────────────────────────────────────────┘ │
                                                         │
AGENT 3: REFLECTION                                     │
┌─────────────────────────────────────────────────────┐ │
│  Input: Query + Retrieved documents                  │ │
│                                                      │ │
│  Processing:                                         │ │
│  1. Evaluate document relevance                     │ │
│  2. Check completeness of information               │ │
│  3. Assess quality and authority                    │ │
│  4. Calculate confidence score                      │ │
│  5. Decide on action                                │ │
│                                                      │ │
│  Output:                                             │ │
│  - sufficient: bool                                  │ │
│  - confidence: float (0-1)                          │ │
│  - reasoning: str                                    │ │
│  - action: keep/expand_query/refine_query           │ │
│  - suggested_refinement: str (optional)             │ │
│                                                      │ │
│  Self-Correction Loop:                              │ │
│  If not sufficient -> Trigger re-retrieval          │ │
│  Max iterations: 2                                   │ │
└─────────────────────────────────────────────────────┘ │
                                                         │
AGENT 4: RESPONSE SYNTHESIS                             │
┌─────────────────────────────────────────────────────┐ │
│  Input: Query + Documents + Reflection + History     │ │
│                                                      │ │
│  Processing (Chain-of-Thought):                     │ │
│  Step 1: Understand the Question                    │ │
│    - What is being asked?                           │ │
│    - What information is needed?                    │ │
│                                                      │ │
│  Step 2: Analyze Retrieved Information              │ │
│    - What's relevant in documents?                  │ │
│    - Is it sufficient?                              │ │
│                                                      │ │
│  Step 3: Formulate Response                         │ │
│    - Structure the answer                           │ │
│    - Note limitations                               │ │
│                                                      │ │
│  Step 4: Provide Final Answer                       │ │
│    - Clear, professional language                   │ │
│    - Citations included                             │ │
│                                                      │ │
│  Output:                                             │ │
│  - response: str (markdown formatted)               │ │
│  - sources: List[dict]                              │ │
│  - confidence: float                                │ │
│  - retrieved_doc_count: int                         │ │
└─────────────────────────────────────────────────────┘ │
                                                         │

                                                         │
TRADITIONAL RAG:                                        │
  Query → Embed → Search → Generate → Response          │
  (Single pass, no reflection)                          │
                                                         │
AGENTIC RAG:                                            │
  Query → Analyze → Search → Reflect → [Re-search?]    │
       → Synthesize (CoT) → Response                    │
  (Multi-agent, self-correcting)                        │
